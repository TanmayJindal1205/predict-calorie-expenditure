# Predict Calorie Expenditure

A Machine Learning project for predicting calories burned during physical activity using biometric and physiological features.

## Dataset

- Training Samples: 750,000
- Test Samples: 250,000

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- TensorFlow
- XGBoost
- LightGBM
- Matplotlib
- Seaborn

## Feature Engineering

- BMI
- Heart Rate Zones
- Age Groups
- Interaction Features

## Models Evaluated

- Linear Regression
- ElasticNet
- XGBoost
- LightGBM
- Deep Neural Network

## Results

Best Model: XGBoost

RMSE: 3.60 Calories
