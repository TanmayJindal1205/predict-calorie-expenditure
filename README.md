# Predict Calorie Expenditure

An end-to-end Machine Learning pipeline for predicting calorie expenditure using biometric and physiological data, featuring advanced feature engineering, model benchmarking, and ensemble learning techniques.

## Dataset

- Training Samples: 750,000
- Test Samples: 250,000

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- TensorFlow
- XGBoost
- LightGBM
- Matplotlib
- Seaborn

## Feature Engineering

- BMI
- Heart Rate Zones
- Age Groups
- Interaction Features

## Models Evaluated

- Linear Regression
- ElasticNet
- XGBoost
- LightGBM
- Deep Neural Network

## Results

Best Model: XGBoost

RMSE: 3.60 Calories
